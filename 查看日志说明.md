# 查看 Tauri 开发日志

## 方法一：在前台运行（推荐）

停止后台进程后，在前台运行以查看实时日志：

```bash
cd /Users/chenzhenqiang/Desktop/test/binder-project
npm run tauri:dev
```

这样可以看到：
- Vite 开发服务器的日志
- Rust 编译日志
- Tauri 应用启动日志
- 后端 Rust 代码的 `println!` 输出
- 前端控制台日志（在浏览器开发者工具中）

## 方法二：查看浏览器控制台

前端日志在浏览器开发者工具中：
1. 打开应用窗口
2. 按 `F12` 或 `Cmd+Option+I` (Mac) 打开开发者工具
3. 切换到 "Console" 标签页

## 方法三：查看系统日志（macOS）

```bash
# 查看系统控制台日志
log show --predicate 'process == "binder"' --last 5m

# 或者使用 Console.app
open -a Console
```
