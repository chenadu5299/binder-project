# Build Mode 历史清理说明

旧版 `R-BLD-*` 文档已经从仓库清理完成。

清理原因：

1. 这些文档的中心前提是旧 Build Mode / Discussion Build / Multi-Actor Build。
2. 它们不再承担当前版本任何控制作用。
3. 继续保留会持续干扰当前 `A-CBT-*` 主线的命名、评审与实现判断。

## 当前处理原则

1. 当前与项目级构建相关的唯一生效主线是 `A-CBT-*`。
2. 当前模板 / 执行 / 状态 / 协同问题，应分别回到 `A-CBT-*`、`A-TMP-*`、`A-AG-*`、`A-WS-*` 处理。
3. 若未来重新立项 Discussion Build / Multi-Actor Build，必须从当前 Active 主线重新建文，不得恢复旧 `R-BLD-*`。
4. 如需追溯旧稿，只能查看 Git 历史，不再依赖仓库内保留文档。
