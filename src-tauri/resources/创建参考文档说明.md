# 创建参考 DOCX 文档说明

## 目的

参考 DOCX 文档用于在 HTML → DOCX 转换时保留格式。Pandoc 会使用参考文档中定义的样式来映射 HTML 格式到 DOCX 格式。

## 创建步骤

### 方法 1：使用 Pandoc 从 Markdown 创建（推荐）

1. 使用 Pandoc 将 `reference_template.md` 转换为 DOCX：

```bash
cd src-tauri/resources
pandoc reference_template.md -o reference.docx
```

2. 用 Microsoft Word 打开 `reference.docx`

3. 在 Word 中定义标准样式：
   - **标题 1**：18pt，粗体，Arial，段前段后间距
   - **标题 2**：16pt，粗体，Arial，段前段后间距
   - **标题 3**：14pt，粗体，Arial，段前段后间距
   - **正文**：12pt，Arial，行距 1.5
   - **强调**：12pt，斜体，Arial
   - **列表**：12pt，Arial

4. 保存文件

### 方法 2：直接在 Word 中创建

1. 打开 Microsoft Word
2. 创建新文档
3. 定义标准样式（同上）
4. 保存为 `reference.docx` 到 `src-tauri/resources/` 目录

## 样式定义建议

### 标题样式

- **标题 1**
  - 字体：Arial
  - 字号：18pt
  - 样式：粗体
  - 段前：12pt
  - 段后：6pt

- **标题 2**
  - 字体：Arial
  - 字号：16pt
  - 样式：粗体
  - 段前：10pt
  - 段后：6pt

- **标题 3**
  - 字体：Arial
  - 字号：14pt
  - 样式：粗体
  - 段前：8pt
  - 段后：4pt

### 正文样式

- **正文**
  - 字体：Arial
  - 字号：12pt
  - 行距：1.5 倍
  - 段后：6pt

- **强调**
  - 字体：Arial
  - 字号：12pt
  - 样式：斜体

### 列表样式

- **无序列表**
  - 字体：Arial
  - 字号：12pt
  - 行距：1.5 倍

- **有序列表**
  - 字体：Arial
  - 字号：12pt
  - 行距：1.5 倍

## 验证

创建完成后，参考文档应该：
- 位于 `src-tauri/resources/reference.docx`
- 包含所有标准样式定义
- 可以在 Pandoc 转换时使用

## 使用

参考文档创建后，PandocService 会自动检测并使用它。如果未找到参考文档，会使用默认转换（格式保留可能不完整）。
